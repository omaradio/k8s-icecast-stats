# icecast-stats 

Stats aggregator from Icecast in a k8s cluster to InfluxDb

## Installation

```sh
npm install icecast-stats --save
```



## Dependencies

- [k8s](https://github.com/goyoo/node-k8s-client): Node.js client library for Google&#39;s Kubernetes Kubectl And API

## Dev Dependencies
